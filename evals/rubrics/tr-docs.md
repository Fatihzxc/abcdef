# Rubric — Set 1: Turkish documents (R1/R2/R6-TR)

Score each dimension 1–5 (1 = unusable, 3 = acceptable with edits, 5 = ship as-is).
Record one overall 1–5 score in `score` and the breakdown/reasoning in `note`.

1. **Formal register correctness** — resmî Türkçe; correct courtesy forms,
   no spoken-language slippage, appropriate institutional tone.
2. **Grammar** — spelling, suffixes, agreement, punctuation.
3. **Structural fit** — matches the requested document type (letter / report
   section / procedure / executive summary), with the expected parts present.
4. **Terminology** — uses company/engineering terms consistently and correctly.

**Gate (Step 7):** the set passes only if the mean ≥ **3.5/5** with **no
register-correctness score below 3**. This set carries the project's top risk —
if every candidate fails here, the Turkish risk gate fires.
